⚡ Why Chai CSS

Everything you need,
nothing you don't.

Chai CSS brings the utility-first approach to vanilla HTML with zero build tools. Just include one script and start building.

🪶

Featherweight

Under 6KB of pure JavaScript. No CSS files to load. No bloated bundles. Just raw performance.

⚡

Zero Build Step

No webpack, no PostCSS, no config files. Drop in a script tag and start using utility classes immediately.

🎯

Smart Caching

Styles are computed once and cached. Repeated classes resolve instantly from memory for blazing performance.

MutationObserver watches for DOM changes. Dynamically added elements get styled automatically — perfect for SPAs.

📦 Getting Started

Install in seconds.

Choose your preferred method — npm, CDN, or a simple script tag.

1

npm / yarn

Install the package from npm:

npm install chai-css

Then import in your JS entry file:

require('chai-css'); // or import 'chai-css';
2

CDN / Script Tag

Just add this before your closing </body> tag:

<script src="chai.js"></script>

Or use a CDN (after publishing):

<script src="https://unpkg.com/chai-css"></script>
3

Start Building

Add chai- prefixed classes to any element:

<div class="chai-p-4 chai-bg-blue-500 chai-text-white chai-rounded-lg"> Hello Chai! ☕ </div>
🛠️ How It Works

Utility classes, inline magic.

Prefix any supported utility with chai- and the engine applies the corresponding CSS property as an inline style.

index.html 
<!-- A styled card using only chai-* classes -->
<div class="chai-flex chai-flex-col chai-gap-4 chai-p-8 chai-bg-slate-800 chai-rounded-xl chai-shadow-lg">

  <h2 class="chai-text-2xl chai-font-bold chai-text-white">
    Welcome to Chai ☕
  </h2>

  <p class="chai-text-base chai-text-slate-400 chai-leading-relaxed">
    Style your HTML with utility classes.
    No CSS files needed!
  </p>

  <button class="chai-p-3 chai-bg-violet-500 chai-text-white chai-rounded-md chai-font-semibold chai-cursor-pointer">
    Get Started →
  </button>

</div>
